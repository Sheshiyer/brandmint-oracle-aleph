# Journey and Success Criteria

## End-to-end journey
1. Upload Product MD
2. Review extraction summary
3. Correct/confirm structured fields
4. Complete brand-config wizard
5. Export `brand-config.yaml`
6. Launch handoff to execution pipeline

## North-star outcomes
- Time to first usable config <= 10 minutes
- First-run completion rate >= 70%
- Manual correction ratio <= 25%
- Retry recovery success >= 90%

## UX quality criteria
- Every step has clear heading, purpose, and next action
- Error messages include cause + fix + retry action
- Non-technical language grade: <= 9th grade readability target
- Interaction latency for key actions <= 300ms UI response

## Accessibility baseline
- WCAG AA contrast for text and UI controls
- Full keyboard navigation for all wizard steps
- Visible focus states and semantic headings
