# Prompt Templates: Brand Product Prompter

All templates use `[BRAND NAME]` variable substitution. Replace with target brand before using.

---

## Section 1: Core Product Concepts

### 1.1 Creative Product Concepts

```
[BRAND NAME]: 
A high-end, glossy concept art magazine editorial photograph of a unique, unexpected functional object conceptualized and designed by the brand.

1. The Concept & Object (AI Invention):
Based on the design philosophy, heritage, and material vocabulary of the specified brand, the AI must invent a novel utility product (NOT standard clothing, shoes, or bags). Examples could be home goods, tech accessories, tools, or sporting equipment, reinterpretated through the brand's lens. The object should feel sculptural yet functional.

2. Materials & Details (Hyper-Premium):
The object is constructed from ultra-premium, highly tactile materials characteristic of the brand (e.g., patinated exotic leathers, brushed aerospace-grade titanium, sculpted matte ceramics, molded carbon fiber, or technical high-fashion textiles). Every detail is hyper-realistic: visible stitching, microscopic material grain, precision engravings, and complex texture contrasts.

3. Photography & Lighting (Cinematic Studio):
Shot on a medium format Phase One camera with a 100mm macro lens. Extremely shallow depth of field, with sharp focus on the hero details of the object and a creamy, smooth bokeh background. The lighting is sophisticated studio softbox lighting.

4. Environment:
A seamless, impeccably clean studio cyclorama background in a pure, ultra-light pastel tone, free of shadows.

5. Layout & UI Elements (Strict Placement):
- Bottom Right Corner: A small, understated, monochrome gray logo of the brand.
- Bottom Left Corner: Small, minimalist monochrome gray text describing the invented product. Font: Manrope Regular with tight tracking.
```

**Variables:** `[BRAND NAME]`
**Best for:** Speculative product invention, editorial concept art
**Camera:** Phase One medium format, 100mm macro, shallow DOF

### 1.2 Premium Beverage

```
[BRAND NAME] is launching a new functional wellness elixir. As Creative Director, devise a product name and visualize a complete high-end promotional shot. Aesthetic: "Cosmic Premium" -- technological, clean, sophisticated.

THE PRODUCT: Sculptural multi-layered beverage bottle, futuristic form. Materials: bead-blasted titanium, frosted borosilicate glass, textured haptic polymer grips. Liquid must have distinct natural color relevant to function. Subtle minimalist technical typography in matte white ink on glass (aerospace labeling aesthetic).

THE ENVIRONMENT: Seamless studio. Background: solid clean light pastel tone complementing liquid color. Ultra-soft diffused studio lighting.

PHOTOGRAPHY: 100mm macro, shallow DOF, sharp focus on textures and printed graphics. 8k, hyper-realistic.

GRAPHIC OVERLAYS: Dark gray UI elements. Bottom Left: small Manrope Regular text. Bottom Right: small dark gray logomark.
```

**Variables:** `[BRAND NAME]`
**Best for:** Beverage product launches, wellness brand extensions
**Camera:** 100mm macro, shallow DOF, 8K

### 1.3 Branded Souvenirs (Capsule Collection)

```
[BRAND NAME], conceptualized as a cohesive, premium "capsule collection" of branded merchandise and functional souvenirs. Curated set of various physical objects (tech accessories, utility gear, lifestyle items, stationery) semantically relevant to the brand's industry. Every item uses brand's official color palette with bold logo integration. Clean organized group composition (knolling or isometric) on seamless infinite studio white background. Soft professional lighting, subtle shadows. Wide framing with significant negative space.

**Variables:** `[BRAND NAME]`

**Composition:** Knolling or isometric group layout

### 1.4 Branded Souvenirs (Creative Concepts)

```
[BRAND NAME].
Act as a creative director curating "recontextualized everyday objects."
DIVERSE OBJECT SELECTION: Randomly select ONE object from: Analog Office, Sport & Leisure, Barware & Kitchen, Utility & Tools, Grooming, Travel.
THE CONCEPT: Elevate mundane object into luxury artifact using brand's signature materials and colors.
MATERIALS: Marble, brushed titanium, matte rubber, carbon fiber, gold plating, high-grade leather. Branding physically integrated (embossed/engraved).
PRESENTATION: No boxes. Object alone on pedestal, fully exposed. Knolling if multiple parts.
PHOTOGRAPHY: Wallpaper*/Dezeen style. Soft high-key lighting. Clean white/grey cyclorama.
GRAPHIC OVERLAYS: Bottom Left: 2-line text block, Manrope Regular, #414141. Bottom Right: monochrome logomark, #414141.
```

**Variables:** `[BRAND NAME]`
**Best for:** Luxury reinterpretation of everyday objects, design editorial
**Style:** Wallpaper*/Dezeen editorial aesthetic

### 1.5 Viral Product Designer (System Prompt)

```
SYSTEM PROMPT: The Viral Product Architect
ROLE: World-class Concept Art Director & Product Designer (Pentagram, IDEO, DONDA). Specialty: "Speculative Design" and "Hype-culture artifacts."
OBJECTIVE: User provides [BRAND NAME]. Invent a hyper-realistic, unexpected physical product concept.
CORE ALGORITHM:
1. Analyze Brand DNA: Extract iconic materials, colors, textures
2. Select Unexpected Object: Mundane/tech object unrelated to brand
3. Apply Texture Mapping: Wrap object in Brand DNA
4. Viral Check: "Would Highsnobiety or Hypebeast repost this?"
VISUAL STYLE: Ultra Hi-End Product Design. Minimalist light studio. Softbox lighting. Telephoto lens 50-85mm, shallow DOF.
OUTPUT: Single optimized image prompt for Imagen/Midjourney.
```

**Variables:** `[BRAND NAME]`
**Best for:** Generating viral product concepts dynamically
**Note:** This is a SYSTEM PROMPT, not an image prompt. Use it to generate image prompts.

### 1.6 Dieline-to-3D Visualization

```
Assemble the dieline into a perfectly folded 3D box with accurate panel placement, clean edges, and undistorted typography. Preserve all artwork exactly as printed. Render in minimal high-end studio setup on soft neutral background with gentle diffused lighting, subtle shadows, no extra props. Show box upright in refined 3/4 angle. Ultra-realistic detail, true colors, matte paperboard texture, crisp folds, premium editorial aesthetic.
```

**Variables:** None (uses uploaded dieline image)
**Best for:** Converting flat packaging dielines into 3D product renders
**Input:** Upload dieline artwork alongside this prompt

### 6.3 Branded Crumpled Stickers

```
Sharp top-down macro photograph of a single heavily crumpled circular vinyl sticker featuring [BRAND NAME] logo.
MATERIAL: Durable slight-gloss vinyl (not paper). Severely crinkled with sharp ridges and deep stress folds. Specular highlights on fold peaks.
COLOR: Background color = brand's primary color. Logo in standard contrasting color.
LIGHTING: Bright clean high-key overhead white studio. Strong highlights on ridges, crisp shadows in valleys.
BACKGROUND: Strict top-down flat lay. Pure white plastic surface with fine granular texture. Entire sticker in sharp focus.
```

**Variables:** `[BRAND NAME]`
**Best for:** Brand marketing assets, sticker mockups
**Angle:** Strict top-down macro

### 6.4 Premium Beverage Package Design

```
[BRAND NAME]. Ultra-high-end editorial shot of luxury beverage bottle.
VESSEL: Matte white ceramic body with powdery texture. Cap in brand's primary color (matte/satin).
BACKGROUND: Seamless studio cyclorama in brand's primary color. Sophisticated white studio lighting.
BRANDING: Central Gothic/Blackletter text with ultra-subtle micro-embossing (0.2-0.3mm depth) in brand's primary color.
TYPOGRAPHY: Dark grey/black auxiliary text -- "LIMITED EDITION // BATCH 001", flavor name, "500ML", ingredients micro-print.
CORNER ELEMENTS: Top Right: small Gothic brand name. Bottom Right: small original logo.
