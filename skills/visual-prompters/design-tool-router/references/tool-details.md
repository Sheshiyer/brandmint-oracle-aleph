# Design Tool Details

## 1. Logo Diffusion
**URL:** logodiffusion.com
**What:** AI logo generator with multiple input modes

### Capabilities
| Mode | Input | Output |
|------|-------|--------|
| Text to Logo | Text prompt describing desired logo | Up to 4 logo options per prompt |
| Logo to Logo | Existing logo + text prompt for redesign | Redesigned logo variations |
| 2D to 3D | Any 2D logo or image | Stunning 3D illustration |
| Image to Logo | Any photograph or image | Simplified logo or stylized illustration |

### Key Features
- **Content Styles:** Pre-built style guides to improve prompting results
- **AI to Vector:** Convert generated logos to clean, industry-standard vector files
- **Remove Background:** Get transparent PNG output
- **In-Browser Editor:** Edit without Photoshop/Illustrator (coming soon)
- **Infinite regeneration:** Generate new designs unlimited times until satisfied

### Best Practices
- Use Content Styles to guide the AI toward specific aesthetics
- Generate 4 options per prompt, iterate on the best one with Logo-to-Logo
- Convert final selection to vector before using in production
- Use 2D-to-3D for social media variations of an established logo

## Tool Selection Flowchart

```
What do you need?
├── A logo
│   ├── From scratch → Logo Diffusion (Text to Logo)
│   ├── Redesign existing → Logo Diffusion (Logo to Logo)
│   ├── Make it 3D → Logo Diffusion (2D to 3D)
│   └── From a photo → Logo Diffusion (Image to Logo)
├── A video
│   ├── Product demo / screen recording → Phia
│   └── Animated social content → Mojo
├── Design assets
│   ├── 3D objects → Items Plus (3D Render)
│   ├── Abstract shapes → Items Plus (Abstract)
│   ├── Backgrounds → Items Plus (Backgrounds)
│   ├── Icons → Items Plus (Icons)
│   └── Specific finish (glass/chrome/matte) → Items Plus (filter by finish)
└── Not sure → Describe the task, route from there
```
