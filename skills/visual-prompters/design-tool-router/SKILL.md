# Design Tool Router

Meta-routing skill for selecting the right AI design tool based on task requirements. Covers logo generation, screen recording, social media animation, and asset sourcing.

## When to Use

- Unsure which AI tool to use for a design task
- Need to generate or iterate on logos
- Need cinematic screen recordings for product demos
- Need animated social media content (stories, posts)
- Need pre-made 3D/abstract design assets

## Tool Decision Matrix

| Task | Tool | Why |
|------|------|-----|
| Generate logo from text description | **Logo Diffusion** | Text-to-logo with 4 options per prompt |
| Redesign/iterate existing logo | **Logo Diffusion** | Logo-to-logo transformation |
| Convert 2D logo to 3D illustration | **Logo Diffusion** | Native 2D-to-3D conversion |
| Convert image to logo/illustration | **Logo Diffusion** | Image-to-logo simplification |
| Record product demo video | **Phia** | Auto-zoom, motion blur, cinematic transitions |
| Create animated Instagram stories | **Mojo** | 100s of animated templates, text effects |
| Create animated social posts | **Mojo** | Business-focused, professional quality |
| Source 3D rendered assets | **Items Plus** | 700+ assets with Midjourney prompts |
| Find abstract backgrounds | **Items Plus** | 500+ abstract assets, multiple finishes |
| Get isometric illustrations | **Items Plus** | 11+ isometric asset sets |
| Find icon sets | **Items Plus** | 42+ icon asset collections |

## Quick Start

1. **Describe your task** (e.g., "I need a logo for a coffee brand")
2. **Check the matrix above** to find the right tool
3. **Read tool details** in `references/tool-details.md` for capabilities and limitations
4. **Execute** using the recommended tool

See `references/tool-details.md` for detailed capabilities, pricing, and usage notes.
